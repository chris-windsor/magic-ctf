# MAGIC CTF

## Prerequisites

- MongoDB
- Node.JS v8.12+

## Installation / Running

+ Development:
  - Start MongoDB server (`mongod`)
  - `npm i`
  - `npm run dev`
  
+ Production:
  - Start MongoDB server (`mongod`)
  - `npm i`
  - `npm run generate`
  - `npm run start`
