# MAGIC CTF

## ✔️ Prerequisites

- MongoDB
- Node v12

## 💻 Installation / Running

+ Development:
  - Start MongoDB server
  - `npm run dev`
  
+ Production:
  - Start MongoDB server
  - `npm run generate`
  - `npm run start`
